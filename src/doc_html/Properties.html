TMCM610_motoraxis Class

Revision: - Author: filip.lindau



Device Properties
Property name
Property type
Description
TMCM610_device Tango::DEV_STRING Tango name of the TMCM610 controller device
Axis Tango::DEV_SHORT Number of the axis to control, in the range 0-5.
Sleep_before_last_read Tango::DEV_DOUBLE Motor instability time. Time in ms to wait after motor stop to read last position.
Motor_current Tango::DEV_DOUBLE Maximum current (peak) in A sent to the motor. The motor rating is usually in RMS, so this value x sqrt(2) should be used here. Must be below 1.5 A.
Limit0Enable Tango::DEV_BOOLEAN Enabling of limit switch 0.
Limit1Enable Tango::DEV_BOOLEAN Enabling of limit switch 1.
LimitPolarity Tango::DEV_BOOLEAN Polarity of motor limit switches (normally open/closed).
